Class: ChartInteractions

ChartInteractions

This class allocates and maintains and destroys resources for user interactions on the chart.

new ChartInteractions(element, primaryMeasure, configuration, parser, styles)

Construct a ChartInteractions object

Parameters:
Name Type Description
element Object

the DOM element containing the svg that the chart is rendered to
primaryMeasure String

the measure that will serve as the basis for any adaptive aspects of the interface
configuration ChartConfiguration

the ChartConfiguration that configures the chart that these interactions will be connected to
parser DataParser

the parser that parsed the data in the chart
styles ChartStyles

the ChartStyles that style this chart
